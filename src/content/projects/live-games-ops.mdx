**문제**  라이브 서비스 → 장애 감지·비용 통제가 일상. 비용 추이가 안 보이면 낭비를 놓치고, 감지가 늦으면 이용자 이탈.

**접근**  운영을 사람 손에 의존하기보다 **모니터링·비용 가시화를 자동화**해 반복 업무를 줄이고, 종료 단계까지 표준 절차로 마무리.

## 핵심 작업

- **모니터링** — Grafana + CloudWatch 연동 실시간 장애 감지 시스템 구축
- **비용 자동화** — Python 기반 일일 AWS 비용 시각화·자동 메일 공유 시스템 직접 개발
- **리소스 최적화** — AWS Compute Optimizer 권고 활용, Jenkins 배포 자동화
- **라이프사이클 운영** — 런칭~종료 전체 운영, 리소스 백업·안정적 종료 절차 수립

## 성과

- 비용 시각화 자동화로 프로젝트별 비용 추이 **실시간 파악**
- 런칭~종료 **전체 라이프사이클 독립 운영** 경험 확보
